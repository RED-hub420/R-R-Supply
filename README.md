# R&R Pipe and Supply Static Site

Static multi-page website for GitHub Pages.

## Pages
- `index.html` — home
- `inventory.html` — stock list
- `product.html` — item detail
- `services.html` — services
- `cart.html` — material list
- `checkout.html` — request pricing
- `about.html` — company overview
- `contact.html` — contact page

## Notes
- Static site for GitHub Pages.
- The custom domain file is included as `CNAME`.
- Client-side request actions are preview interactions only until a live form handler is connected.
