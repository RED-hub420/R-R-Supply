# R&R Pipe and Supply Demo Site

Static multi-page PVF and fencing supply demo built from the Old Grumpy's structure and reworked into a more industrial quote-first experience.

## Pages
- `index.html` — homepage
- `inventory.html` — searchable inventory catalog
- `product.html` — item detail template
- `services.html` — service overview
- `cart.html` — quote basket
- `checkout.html` — quote request page
- `about.html` — company overview
- `contact.html` — contact and quote form page

## Notes
- This is a static demo intended for GitHub Pages or other static hosting.
- Inventory items are demo records created to fit the R&R Pipe and Supply brand direction.
- Forms are non-live and only simulate a completed quote request.
- Quote basket data is stored in local browser storage.

## Deploy
Upload the folder contents to the root of a GitHub Pages repository.
